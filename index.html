Introduction to PASS ICT283


Welcome to the PASS ICT283 Data Structures & Abstractions companion site.

This companion site is a self-service learning tool intended for Murdoch University Undergraduates attending ICT283 Peer-Assisted Study Sessions(PASS).

Join the community to collaborate with fellow classmates and speed up your learning. Practice building, running and debugging code yourself. You may also undertake additional C++ training to hone your skills.

The 'resources' section of this site contains topic reviews and code practices to strengthen your foundational skills in C++ programming. The coding examples serve as fundamental building blocks and key OOP concepts are demonstrated through simple programs.

Practicing on these code samples will prepare you for the more complex coding assignments ahead.

Note that only the most fundamental data structure concepts are covered here.
Any further learning is up to you.

Expand your knowledge by using the internet to your advantage and attempt all coding exercises and labs in ICT283.

Remember, repetition is the mother of skill. Keep practicing and be consistent. Your coding skills will improve as you build experience.

Thank you for starting.
